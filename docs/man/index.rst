Resiliparse User Manual
+++++++++++++++++++++++

ChatNoir Resiliparse is a collection of robust and fast processing tools for parsing and analyzing web archive data. Resiliparse is part of the `ChatNoir web analytics toolkit <https://github.com/chatnoir-eu/>`_ developed by the `Webis Group <https://webis.de>`_.
